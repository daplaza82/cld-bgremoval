# Cloudinary BG Remove
Use a Netlify custom function to remove background from an image.

## Form 
Capture URL to remove background, public id and tag.

## Netlify Deploy

Set up an account on Netlify and link to a Github repo.

```bash
netlify deploy 
netlify deploy --prod
```
